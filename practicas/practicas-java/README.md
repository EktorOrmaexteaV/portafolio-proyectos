# Mis Prácticas de Programación en Java

En esta carpeta, encontraremos mis proyectos que he desarrollado en Java. Recordando mis conocimientos en el lenguaje de programación Java, explorando sus características, la Programación Orientada a Objetos (POO) y el manejo de estructuras de datos y archivos.

Interpreté el proyecto "Cajero" que hice en C++ al lenguaje de programación Java y usando entornos GUI.

## Proyectos Incluidos

Aquí encontrarás los proyectos de C++ que he desarrollado. Cada proyecto está en su propia subcarpeta y cuenta con un `README.md` detallado para su comprensión y uso.

### [Simulador de Cajero Automático](Cajero/)
* **Descripción**: Rendición del proyecto "Cajero" que hice en C++, usando bases de datos MySQL, al lenguaje de programación Java y usando entornos GUI. Usando la misma estructura de ingreso, alta de clientes y gestión de cuentas, además añadi una opción para imprimir el historial de transacciones a un archivo PDF. 
* **Tecnologías Clave**: Java, Programación Orientada a Objetos (POO), Manejo de Bases de Datos (MySQL), Estructuras de Control, Manejo de Archivos (fstream), Manejo de Entornos GUI (Swing).
* **Aprendizajes Destacados**:
    * Recordatorio de programación en Java.
    * POO en Java.
    * Manejo de bases de datos MySQL en Java.
    * Manejo de entornos GUI en Java.
* **Estado**: Completo (versión mejorada).
